Airbnb เป็น startup ที่ก่อตั้งขึ้นเมื่อปี 2008 ใน San Francisco และได้ขยายตัวอย่างรวดเร็ว ปัจจุบันนี้ Airbnb ได้ให้บริการมากกว่า 190 ประเทศทั่วโลก และได้กลายเป็นหนึ่งในตัวเลือกที่ได้รับความนิยมอย่างมากในหมู่นักเดินทาง ในฐานะที่เป็นอีกหนึ่งทางเลือกนอกเหนือไปจากการเช่าโรงแรมตามปกติซึ่งมีค่าใช้จ่ายที่ค่อนข้างสูง ผู้เช่าสามารถจองห้องพักจาก Airbnb ได้ทั้งจากบนสมาร์ทโฟนและบนเว็บไซต์

หากเจ้าของห้องพักมีห้องอยากให้เช่ากับทาง Airbnb หรือแม้กระทั่งกับทางแพลตฟอร์มอื่น ๆ การที่เจ้าของห้องพักทราบว่าควรตั้งราคาค่าเช่าเท่าไรเป็นสิ่งที่สำคัญ เพราะจะทำให้เจ้าของห้องพักสามารถตั้งราคาที่ทำให้ได้กำไร และในขณะเดียวกันก็ไม่สูงเกินไปจนไม่มีผู้เช่ามาใช้บริการ

งานชิ้นนี้ได้ใช้ Big Data ของทาง Airbnb ในการพัฒนา model ประเภท Linear Regression และ Support Vector Regression (SVR) เพื่อใช้ทำนายค่าเช่าของห้องพัก นอกจากนี้ยังได้ทำการวิเคราะห์ข้อมูลเพื่อให้ได้ความเข้าใจเชิงลึก (insights) อีกด้วย

เจ้าของห้องพักสามารถใช้ประโยชน์จากงานชิ้นนี้ทั้งจาก insights และจากตัว models ทำนายค่าเช่าเพื่อช่วยตัดสินใจหากต้องการต่อเติมห้องน้ำ, เพิ่มเตียงนอน, เพิ่มจำนวนแขกที่รองรับได้ หรือแม้กระทั่งการพิจารณาทำเลที่ต้องการจะซื้ออสังหาริมทรัพย์ว่ามีศักยภาพทางราคาค่าเช่ามากน้อยเพียงใด

# Executive Summary
---

1. งานชิ้นนี้ได้พัฒนา models ทำนายค่าเช่าเพื่อเป็นอีกหนึ่งตัวช่วยให้แก่เจ้าของห้องพักในการตั้งราคาค่าเช่าที่ทำให้ได้กำไร และในขณะเดียวกันก็ไม่สูงเกินไปจนไม่มีผู้เช่ามาใช้บริการ

2. ตัว models สามารถปรับเปลี่ยนปัจจัยต่าง ๆ ได้แก่ เมืองของห้องพัก, ประเภทของห้อง, จำนวนแขกที่รองรับได้, จำนวนเตียง, และจำนวนห้องน้ำ เพื่อทำนายผลลัพธ์ออกมาเป็นราคาค่าเช่า
3. งานชิ้นนี้ได้พัฒนา Regression Models ประเภท Linear Regression และ Support Vector Regression (SVR) เพื่อทำนายค่าเช่าจาก Big Data โดย model ที่ทำนายได้แม่นยำสุดคือ Linear Regression โดยมีค่า Root Mean Square Error (RMSE) เท่ากับ 46 USD และค่า R-squared เท่ากับ 0.52 สำหรับรายละเอียดจะได้อธิบายในส่วนของ [Models ทำนายค่าเช่า](#bullet2) 
4. งานชิ้นนี้ได้ทำการวิเคราะห์ข้อมูล Exploratory Data Analysis (EDA) เพื่อให้ได้ความเข้าใจเชิงลึก (insights) โดยสรุปคือ

    - เมือง Manhattan และ Brooklyn มีค่าเช่าโดยทั่วไป (ค่ากลาง) สูงเป็นอันดับต้น ๆ คือ 129 และ 92 USD ตามลำดับ
    
    - ห้องประเภทห้องโรงแรมและเช่าทั้งหลัง มีค่าเช่าโดยทั่วไป (ค่ากลาง) สูงเป็นอันดับต้น ๆ คือ 201 และ 149 USD ตามลำดับ
    - หากต้องการคิดค่าเช่าสูง ๆ สามารถเพิ่มจำนวนเตียง หรือเพิ่มจำนวนแขกที่รองรับให้มากขึ้น
    - ทั้งจำนวนคืนขั้นต่ำและจำนวนคืนสูงสุด ไม่ส่งผลต่อราคาค่าเช่า
    - ห้องส่วนใหญ่รองรับแขกได้ 1-2 คน (65% ของห้องทั้งหมด), มี 1-2 เตียง (90% ของห้องทั้งหมด), และมี 1 ห้องน้ำ (87% ของห้องทั้งหมด)

# Models ทำนายค่าเช่า
---

งานชิ้นนี้ได้ใช้ Big Data ของทาง Airbnb ในการพัฒนา model ประเภท Linear Regression และ Support Vector Regression (SVR) เพื่อใช้ทำนายค่าเช่าของห้องพัก

สำหรับตัว models ท่านสามารถดูได้ในส่วนของ [Jupyter Notebooks](#bullet6) ในตอนท้าย

Model ที่ทำนายได้แม่นยำสุดคือ Linear Regression โดยมีค่า Root Mean Square Error (RMSE) เท่ากับ 46 USD และค่า R-squared เท่ากับ 0.52 ซึ่งถือว่าทำนายได้ดีในระดับปานกลางเนื่องจากชุดข้อมูลนี้มีความผันผวนสูงและมีสัดส่วนข้อมูลที่ไม่สมดุล

สำหรับปัญหาสัดส่วนข้อมูลที่ไม่สมดุล จะได้อธิบายในส่วนถัดไปคือ Exploratory Data Analysis ซึ่งจะได้แสดงไว้ใน[แผนภาพวงกลมส่วนแบ่งของประเภทห้อง](#bullet_a)และ[เมืองของห้องพัก](#bullet_b) และสำหรับแนวทางการแก้ปัญหาจะได้อธิบายไว้ในส่วนของ [Future Scopes](#bullet5) ในตอนท้าย

# Exploratory Data Analysis เพื่อให้ได้ Insights
---
งานชิ้นนี้ได้ทำการวิเคราะห์ข้อมูลเพื่อให้ได้ความเข้าใจเชิงลึก (insights) โดยอภิปรายได้ดังต่อไปนี้

### 1) ราคาค่าเช่าแยกตามแต่ละเมืองของห้องพัก
![https://raw.githubusercontent.com/ProStepSky/ProStepSky.github.io/master/img/1_Airbnb/1.png](https://raw.githubusercontent.com/ProStepSky/ProStepSky.github.io/master/img/1_Airbnb/1.png)

1. เมือง Manhattan และ Brooklyn มีค่าเช่าแปรปรวนมากที่สุด (บางห้องมีค่าเช่าแพงมาก บางห้องมีค่าเช่าถูกมาก)
2. เมือง Bronx, Brooklyn, Manhattan, Queens, และ Staten Island มีค่าเช่าโดยทั่วไป (ค่ากลาง) ตามลำดับดังนี้ 72, 92, 129, 70, และ 86 USD

หากเจ้าของห้องพักมีห้องพักอยู่แล้ว สามารถทราบได้ว่าห้องพักที่ตั้งอยู่ในเมืองนี้ มีค่าเช่าโดยทั่วไป (ค่ากลาง) เป็นเท่าไร หรือหากดูขนาดของกล่องของ box plot ซึ่งคิดเป็น 50% ของจำนวนห้องทั้งหมด ก็จะทำให้ทราบว่าห้องส่วนใหญ่ในเมืองนี้ตั้งราคาอยู่ในช่วงใด

แต่หากเจ้าของห้องพักต้องการจะซื้ออสังหาริมทรัพย์จากทำเลที่มีศักยภาพเพื่อปล่อยให้เช่าในอนาคต อาจลองพิจารณาเมือง Manhattan หรือ Brooklyn เนื่องจากมีแนวโน้มว่าจะได้ค่าเช่าสูงและทำกำไรได้


### 2) ราคาค่าเช่าแยกตามแต่ละประเภทห้อง

![https://raw.githubusercontent.com/ProStepSky/ProStepSky.github.io/master/img/1_Airbnb/2.png](https://raw.githubusercontent.com/ProStepSky/ProStepSky.github.io/master/img/1_Airbnb/2.png)

1. ห้องประเภทเช่าทั้งหลังและห้องโรงแรม มีค่าเช่าที่แปรปรวนสูง (บางห้องมีค่าเช่าแพงมาก บางห้องมีค่าเช่าถูกมาก)
2. ห้องประเภทเช่าทั้งหลัง, ห้องโรงแรม, ห้องส่วนตัว, และห้องแชร์ มีค่าเช่าโดยทั่วไป (ค่ากลาง) ตามลำดับดังนี้ 149, 201, 64, และ 48 USD

ข้อมูลเหล่านี้ทำให้ทราบว่า นอกเหนือไปจากห้องโรงแรมแล้ว การปล่อยใช้เช่าได้ทั้งหลังจะมีแนวโน้มได้ค่าเช่าสูงกว่าการปล่อยเช่าแบบห้องส่วนตัวหรือแบบห้องแชร์


### 3) จำนวนประเภทห้องแยกตามแต่ละเมือง

![https://raw.githubusercontent.com/ProStepSky/ProStepSky.github.io/master/img/1_Airbnb/3.png](https://raw.githubusercontent.com/ProStepSky/ProStepSky.github.io/master/img/1_Airbnb/3.png)

1. เมือง Manhattan มีห้องประเภทเช่าทั้งหลังมากกว่าห้องส่วนตัวถึง 76%
2. เมือง Queens มีห้องประเภทห้องส่วนตัวมากกว่าเช่าทั้งหลังถึง 67%
3. เมือง Brooklyn, Staten Island, และ Bronx มีห้องประเภทเช่าทั้งหลังและห้องส่วนตัวใกล้เคียงกัน

### 4) ส่วนแบ่งของประเภทห้อง

![https://raw.githubusercontent.com/ProStepSky/ProStepSky.github.io/master/img/1_Airbnb/4.png](https://raw.githubusercontent.com/ProStepSky/ProStepSky.github.io/master/img/1_Airbnb/4.png)

### 5) ส่วนแบ่งของเมืองของห้องพัก

![https://raw.githubusercontent.com/ProStepSky/ProStepSky.github.io/master/img/1_Airbnb/5.png](https://raw.githubusercontent.com/ProStepSky/ProStepSky.github.io/master/img/1_Airbnb/5.png)

3. อีก 20% ที่เหลือ มาจากเมือง Queens, Bronx, และ Staten Island

ข้อมูลเหล่านี้บ่งบอกว่า หากใช้ model ของงานชิ้นนี้ทำนายห้องพักในเมือง Queens, Bronx, หรือ Staten Island จะทำให้ได้ค่าที่แม่นยำน้อย เนื่องจากข้อมูลที่นำมาใช้สร้าง model มีจำนวนน้อย (สำหรับแนวทางการแก้ปัญหาจะได้อธิบายไว้ในส่วน [Future Scopes](#bullet5) ในตอนท้าย)


### 6) ราคาค่าเช่ากับจำนวนแขกที่รองรับ

1. 91% ของจำนวนห้องทั้งหมด จะรองรับแขกได้ระหว่าง 1-4 คน

![https://raw.githubusercontent.com/ProStepSky/ProStepSky.github.io/master/img/1_Airbnb/6.png](https://raw.githubusercontent.com/ProStepSky/ProStepSky.github.io/master/img/1_Airbnb/6.png)

การได้รู้ว่าส่วนใหญ่ห้องพักรองรับแขกได้กี่คนจะทำให้เจ้าของห้องพักมีข้อมูลเพื่อเปรียบเทียบว่า หากจะกำหนดให้ห้องพักนั้นรองรับแขกได้แค่ 1-2 คนก็ไม่ได้เป็นเลขที่น้อยไป เพราะส่วนใหญ่ (ประมาณ 66%) ก็กำหนดกันไว้ประมาณนี้

2. ยิ่งรองรับแขกได้มากยิ่งมีแนวโน้มว่าค่าเช่ายิ่งสูง

![https://raw.githubusercontent.com/ProStepSky/ProStepSky.github.io/master/img/1_Airbnb/7.png](https://raw.githubusercontent.com/ProStepSky/ProStepSky.github.io/master/img/1_Airbnb/7.png)

หากต้องการคิดค่าเช่าสูง ๆ สามารถปรับปรุงห้องพักให้รองรับแขกได้มากขึ้น

### 7) ราคาค่าเช่ากับจำนวนเตียง

1. 99% ของจำนวนห้องทั้งหมด จะมีจำนวนเตียงระหว่าง 1-4 เตียง

![https://raw.githubusercontent.com/ProStepSky/ProStepSky.github.io/master/img/1_Airbnb/8.png](https://raw.githubusercontent.com/ProStepSky/ProStepSky.github.io/master/img/1_Airbnb/8.png)

2. ยิ่งจำนวนเตียงมากยิ่งมีแนวโน้มว่าค่าเช่ายิ่งสูง

![https://raw.githubusercontent.com/ProStepSky/ProStepSky.github.io/master/img/1_Airbnb/9.png](https://raw.githubusercontent.com/ProStepSky/ProStepSky.github.io/master/img/1_Airbnb/9.png)

หากต้องการคิดค่าเช่าสูง ๆ ก็สามารถเพิ่มเตียงนอนได้ ซึ่งการเพิ่มเตียงนอนทำให้จำนวนแขกที่รองรับสูงขึ้น (จากการวิเคราะห์พบว่าสองปัจจัยนี้มีสหสัมพันธ์กันสูง) จึงส่งผลให้คิดค่าเช่าได้มากขึ้น

### 8) ราคาค่าเช่ากับจำนวนห้องน้ำ

1. 87% ของจำนวนห้องทั้งหมด จะมีห้องน้ำ 1 ห้อง

![https://raw.githubusercontent.com/ProStepSky/ProStepSky.github.io/master/img/1_Airbnb/10.png](https://raw.githubusercontent.com/ProStepSky/ProStepSky.github.io/master/img/1_Airbnb/10.png)

2. จำนวนห้องน้ำไม่มีความสัมพันธ์กับค่าเช่า

![https://raw.githubusercontent.com/ProStepSky/ProStepSky.github.io/master/img/1_Airbnb/11.png](https://raw.githubusercontent.com/ProStepSky/ProStepSky.github.io/master/img/1_Airbnb/11.png)

ถึงแม้ว่าห้องที่คิดค่าเช่าสูง ๆ จะมีจำนวนห้องน้ำที่มาก แต่นั่นไม่ได้หมายความว่าจำนวนห้องน้ำทำให้ราคาเพิ่มขึ้น จำนวนห้องน้ำที่มากสามารถเป็นผลมาจากการที่ห้องพักรองรับแขกได้มาก ซึ่งห้องพักที่รองรับแขกได้มากก็มีแนวโน้มว่าค่าเช่ายิ่งสูง  ถึงแม้การต่อเติมห้องน้ำจะไม่ได้เพิ่มราคาค่าเช่า แต่ก็เพิ่มประสบการณ์ให้แก่ผู้เช่าได้ ถ้าห้องพักมีคนมารีวิวในทางบวก ต่อไปผู้คนก็อยากมาใช้บริการ

# สรุป
---

โดยสรุปแล้ว เจ้าของห้องพักสามารถใช้ประโยชน์จากงานชิ้นนี้ทั้งจาก insights และจากตัว model ทำนายค่าเช่าซึ่งสามารถปรับเปลี่ยนปัจจัยต่าง ๆ ได้ เช่น จำนวนห้องน้ำ, จำนวนเตียงนอน, และอื่น ๆ ซึ่งสามารถช่วยตัดสินใจหากต้องการต่อเติมห้องน้ำ, เพิ่มเตียงนอน, เพิ่มจำนวนแขกที่รองรับได้ หรือแม้กระทั่งการพิจารณาทำเลที่ต้องการจะซื้ออสังหาริมทรัพย์ว่ามีศักยภาพทางราคาค่าเช่ามากน้อยเพียงใด

ไม่เพียงแต่เจ้าของห้องพักเท่านั้นที่จะได้ประโยชน์จาก model ทำนายค่าเช่านี้ ผู้เช่าเองก็สามารถใช้ประโยชน์ทั้งจาก insights และตัว model ทำนายค่าเช่าด้วยเช่นกัน ผู้เช่าสามารถใช้ model นี้ทำนายว่า จากคุณลักษณะของห้องพักที่จะเลือกเช่านั้น มีแนวโน้มว่าตั้งราคาสมเหตุสมผลหรือไม่ หรือหากต้องการเช่าห้องที่อยู่ในทำเลที่มีศักยภาพ ก็จะทำให้ทราบว่าต้องเพิ่มราคามากน้อยเพียงใด หรืออาจลดจำนวนแขกที่รองรับได้, จำนวนห้องน้ำ, และจำนวนเตียงภายในห้อง เพื่อแลกกับการได้เช่าในทำเลที่มีศักยภาพ เป็นต้น

# Future Scopes
---

1. ชุดข้อมูลนี้มีสัดส่วนข้อมูลที่ไม่สมดุล (imbalanced dataset) เนื่องจากมีสัดส่วนประเภทห้องและสัดส่วนเมืองที่ไม่สมดุล ดังที่ได้แสดงในแผนภาพวงกลมส่วนแบ่งของประเภทห้องและเมืองของห้องพักในส่วน Exploratory Data Analysis สำหรับปัญหาสัดส่วนข้อมูลที่ไม่สมดุลนี้เป็นปัญหาที่พบได้บ่อยในการเก็บข้อมูล แนวทางแก้ไขคือหากสามารถเก็บข้อมูลดังต่อไปนี้ให้มีจำนวนมากขึ้นได้:
    - ห้องพักที่เป็นประเภทห้องแชร์และห้องโรงแรม
    - ห้องพักที่มาจากเมือง Queens, Bronx, และ Staten Island
    
    จะทำให้บรรเทาปัญหาสัดส่วนข้อมูลที่ไม่สมดุลได้ และจะส่งผลให้ models ทำนายได้แม่นยำยิ่งขึ้น

2. อาจลองนำค่า latitude และ longitude มาใช้ร่วมด้วย โดยมีสมมติฐานว่าห้องพักที่เดินทางสะดวก เช่น ใกล้สนามบิน อาจมีค่าเช่าที่สูงกว่า อาจทำให้ model ทำนายได้แม่นยำยิ่งขึ้น
3. การทำนายข้อมูลในครั้งนี้ เป็นข้อมูลเชิงปริมาณ (quantity) ซึ่งใช้ Logistic Regression และ Support Vector Regression ในการทำนาย เนื่องจากข้อมูลชุดนี้มีความผันผวนสูงและมีสัดส่วนข้อมูลที่ไม่สมดุล ในอนาคตอาจลองใช้ Deep Learning เพื่อดูว่าทำให้ model ทำนายได้แม่นยำยิ่งขึ้นอีกหรือไม่
