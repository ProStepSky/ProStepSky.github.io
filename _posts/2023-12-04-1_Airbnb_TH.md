Airbnb เป็น startup ที่ก่อตั้งขึ้นเมื่อปี 2008 ใน San Francisco และได้ขยายตัวอย่างรวดเร็ว ปัจจุบันนี้ Airbnb ได้ให้บริการมากกว่า 190 ประเทศทั่วโลก และได้กลายเป็นหนึ่งในตัวเลือกที่ได้รับความนิยมอย่างมากในหมู่นักเดินทาง ในฐานะที่เป็นอีกหนึ่งทางเลือกนอกเหนือไปจากการเช่าโรงแรมตามปกติซึ่งมีค่าใช้จ่ายที่ค่อนข้างสูง ผู้เช่าสามารถจองห้องพักจาก Airbnb ได้ทั้งจากบนสมาร์ทโฟนและบนเว็บไซต์

หากเจ้าของห้องพักมีห้องอยากให้เช่ากับทาง Airbnb หรือแม้กระทั่งกับทางแพลตฟอร์มอื่น ๆ การที่เจ้าของห้องพักทราบว่าควรตั้งราคาค่าเช่าเท่าไรเป็นสิ่งที่สำคัญ เพราะจะทำให้เจ้าของห้องพักสามารถตั้งราคาที่ทำให้ได้กำไร และในขณะเดียวกันก็ไม่สูงเกินไปจนไม่มีผู้เช่ามาใช้บริการ

งานชิ้นนี้ได้ใช้ Big Data ของทาง Airbnb ในการพัฒนา model ประเภท Linear Regression และ Support Vector Regression (SVR) เพื่อใช้ทำนายค่าเช่าของห้องพัก นอกจากนี้ยังได้ทำการวิเคราะห์ข้อมูลเพื่อให้ได้ความเข้าใจเชิงลึก (insights) อีกด้วย

เจ้าของห้องพักสามารถใช้ประโยชน์จากงานชิ้นนี้ทั้งจาก insights และจากตัว models ทำนายค่าเช่าเพื่อช่วยตัดสินใจหากต้องการต่อเติมห้องน้ำ, เพิ่มเตียงนอน, เพิ่มจำนวนแขกที่รองรับได้ หรือแม้กระทั่งการพิจารณาทำเลที่ต้องการจะซื้ออสังหาริมทรัพย์ว่ามีศักยภาพทางราคาค่าเช่ามากน้อยเพียงใด

# Executive Summary
---

1. งานชิ้นนี้ได้พัฒนา models ทำนายค่าเช่าเพื่อเป็นอีกหนึ่งตัวช่วยให้แก่เจ้าของห้องพักในการตั้งราคาค่าเช่าที่ทำให้ได้กำไร และในขณะเดียวกันก็ไม่สูงเกินไปจนไม่มีผู้เช่ามาใช้บริการ

2. ตัว models สามารถปรับเปลี่ยนปัจจัยต่าง ๆ ได้แก่ เมืองของห้องพัก, ประเภทของห้อง, จำนวนแขกที่รองรับได้, จำนวนเตียง, และจำนวนห้องน้ำ เพื่อทำนายผลลัพธ์ออกมาเป็นราคาค่าเช่า
3. งานชิ้นนี้ได้พัฒนา Regression Models ประเภท Linear Regression และ Support Vector Regression (SVR) เพื่อทำนายค่าเช่าจาก Big Data โดย model ที่ทำนายได้แม่นยำสุดคือ Linear Regression โดยมีค่า Root Mean Square Error (RMSE) เท่ากับ 46 USD และค่า R-squared เท่ากับ 0.52 สำหรับรายละเอียดจะได้อธิบายในส่วนของ models ทำนายค่าเช่า 
4. งานชิ้นนี้ได้ทำการวิเคราะห์ข้อมูล Exploratory Data Analysis (EDA) เพื่อให้ได้ความเข้าใจเชิงลึก (insights) โดยสรุปคือ

    - เมือง Manhattan และ Brooklyn มีค่าเช่าโดยทั่วไป (ค่ากลาง) สูงเป็นอันดับต้น ๆ คือ 129 และ 92 USD ตามลำดับ
    
    - ห้องประเภทห้องโรงแรมและเช่าทั้งหลัง มีค่าเช่าโดยทั่วไป (ค่ากลาง) สูงเป็นอันดับต้น ๆ คือ 201 และ 149 USD ตามลำดับ
    - หากต้องการคิดค่าเช่าสูง ๆ สามารถเพิ่มจำนวนเตียง หรือเพิ่มจำนวนแขกที่รองรับให้มากขึ้น
    - ทั้งจำนวนคืนขั้นต่ำและจำนวนคืนสูงสุด ไม่ส่งผลต่อราคาค่าเช่า
    - ห้องส่วนใหญ่รองรับแขกได้ 1-2 คน (65% ของห้องทั้งหมด), มี 1-2 เตียง (90% ของห้องทั้งหมด), และมี 1 ห้องน้ำ (87% ของห้องทั้งหมด)

# Models ทำนายค่าเช่า
---

งานชิ้นนี้ได้ใช้ Big Data ของทาง Airbnb ในการพัฒนา model ประเภท Linear Regression และ Support Vector Regression (SVR) เพื่อใช้ทำนายค่าเช่าของห้องพัก

สำหรับตัว models ท่านสามารถดูได้ในส่วนของ [Jupyter Notebooks](#bullet6) ในตอนท้าย

Model ที่ทำนายได้แม่นยำสุดคือ Linear Regression โดยมีค่า Root Mean Square Error (RMSE) เท่ากับ 46 USD และค่า R-squared เท่ากับ 0.52 ซึ่งถือว่าทำนายได้ดีในระดับปานกลางเนื่องจากชุดข้อมูลนี้มีความผันผวนสูงและมีสัดส่วนข้อมูลที่ไม่สมดุล

สำหรับปัญหาสัดส่วนข้อมูลที่ไม่สมดุล จะได้อธิบายในส่วนถัดไปคือ Exploratory Data Analysis ซึ่งจะได้แสดงไว้ในแผนภาพวงกลมส่วนแบ่งของประเภทห้องและเมืองของห้องพัก และสำหรับแนวทางการแก้ปัญหาจะได้อธิบายไว้ในส่วนของ [Future Scopes](#bullet5) ในตอนท้าย
